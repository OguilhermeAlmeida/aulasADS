O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. É uma linguagem de marcação que utiliza elementos (tags) para estruturar o conteúdo de uma página.

Estrutura Básica de um Documento HTML

Todo documento HTML começa com uma estrutura básica. Aqui está um exemplo simples:

        <!DOCTYPE html>
        <html>
        <head>
            <meta charset="UTF-8">
            <title>Título da Página</title>
        </head>
        <body>
            Conteúdo da Página
        </body>
        </html>

Elementos HTML Básicos

Alguns elementos HTML comuns incluem:

  • <p>: Parágrafo
  • <h1> a <h6>: Títulos
  • <a>: Link
  • <ul> e <li>: Listas não ordenadas
  • <ol> e <li>: Listas ordenadas
  • <img>: Imagem

Comentários em HTML

Você pode adicionar comentários ao seu código HTML usando <!-- Seu Comentário Aqui --!>. Comentários são ignorados pelo navegador e são úteis para documentar seu código.
